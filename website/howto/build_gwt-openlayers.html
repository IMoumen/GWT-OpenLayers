Building the GWT OpenLayers plugin

The GWT OpenLayers plugin is a GWT module that is distributed as a jar. See this tutorial on how to use the plugin. Below are the steps for building the plugin.

Step 1. Install Maven: The GWT OpenLayers plugin comes with a Maven build script. For information on Maven, see the Maven website.

Step 2. Download GWT: You will need the 'gtw-user.jar' that ships with GWT. Currently GWT OpenLayers builds against GWT version 1.4.60. Different versions of GWT can be downloaded here

Step 3. Install the 'gwt-user.jar' in your local repository: Maven can manage dependencies for you. It retrieves dependencies either from your local repository, or from a remote repository. To install 'gwt-user.jar', first rename it to 'gwt-user-VERSION.jar'. After the rename, open a command prompt at the directory where you put the jar and do:

mvn install:install-file -Dfile="gwt-user-1.4.60.jar" -DgroupId=gwt -DartifactId=gwt-user -Dversion=1.4.60 -Dpackaging=jar

Step 4. Install a CVS client: If you use an Eclipse IDE, than you already have a CVS client installed. A commandline CVS client can be downloaded here.

Step 5. Get the code from CVS: Checking out code from a CVS repository is explained here.

Step 6. Package the module as a jar: Open a command prompt at the root directory of the project and do:

mvn package
