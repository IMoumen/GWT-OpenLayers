Introduction to GWT OpenLayers plugin

What is it?

The GWT OpenLayers plugin let's you use OpenLayers in your GWT application. See the user reference fore more information.

Where can I get it?

The latest stable version is version 0.4, which you can get here. All files for download, also older versions of GWT-OpenLayers, can be found here. To get the lastest version of GWT-OpenLayers you will need to check out the source from Mercurial and build the module yourself, or download the latest snapshot. See the developer reference on how to build the module in two easy steps (given that you have the Mercurial and Maven commandline clients installed).

Version 0.4 uses java 1.5 specific features and will only run on GWT 1.5 or up. The recommended version of OpenLayers to use with 0.4 is OpenLayers 2.8. Version 0.2 will run with GWT 1.4 and OpenLayers 2.4 and up, but lacks many features in Gwt-Openlayers 0.4, so upgrading is recommended. Be aware that the package name has changed from version 0.2 to 0.4!

How do I use it?

Currently, the best way to see how GWT-OpenLayers is used is by looking at the showcase source. Check the user reference for information.

GWT-Openlayers wraps OpenLayers and needs OpenLayers to work. Version 0.4 works best with OpenLayers 2.8, which can be downloaded here.
