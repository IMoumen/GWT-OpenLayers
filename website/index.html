Introduction to GWT OpenLayers plugin

What is it?

The GWT OpenLayers plugin let's you use OpenLayers in your GWT application. You can use the GWT OpenLayers 0.2 download out of the box with GWT 1.4 and OpenLayers 2.4 and up. Click here for an introductory tuturial.

The introductory tutorial is no longer valid for GWT OpenLayers 0.4. For GWT OpenLayers 0.4 code examples, take a look at the source code of the Showcase, which can be obtained by checking out the GWT Openlayers source code from Mercurial (see this page).

To use GWT OpenLayers with GWT 1.5 or GWT 1.6 you currently need to check out the source from Mercurial and build the module yourself. See the developer reference on how to build the module in two easy steps (given that you have the Mercurial and Maven commandline clients installed). For a first, limited showcase, see this GWT OpenLayers showcase. GWT Openlayers 0.4 works best with OpenLayers 2.7, which can be downloaded here.
