GWT OpenLayers plugin

Introduction to GWT OpenLayers plugin

What is it?

The GWT OpenLayers plugin let's you use OpenLayers in your GWT application. You can use the GWT OpenLayers 0.2 download out of the box with GWT 1.4 and OpenLayers 2.4 and up.

Click here for an introductory tuturial.

To use the GWT OpenLayers with GWT 1.5 you currently need to check out the source from CVS and build the module yourself. For a first, limited showcase, see this GWT OpenLayers showcase.
